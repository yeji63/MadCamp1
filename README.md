# MadCamp1
윤예지, 류수현

## 프로젝트 이름 생각하기
탭 3개의 안드로이드 앱

### TAB 1
핸드폰 연락처와 연동하여 연락처 구축. 추가, 검색 기능.
연락처 추가하면 실제 핸드폰 연락처에도 저장

### TAB 2
갤러리 구축. 종류별 앨범 생성하여 앨범 눌러서 사진 볼 수 있음. 사진 누르면 크게 볼 수 있음.

### TAB 3
슬라이드 NxN 퍼즐 게임
사용자가 이미지와 N 선택 가능.
빈칸으로 퍼즐 조각 밀어서 원래 이미지 완성시키기

기능. 함수형태 코드 한줄만 뭐 이런식
